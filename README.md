<img width="792" height="507" alt="image" src="https://github.com/user-attachments/assets/a41356ff-3475-44c1-adde-a0e886f7b220" />Wanderlust is a full-stack MERN web application designed to provide a seamless hotel listing experience across different cities worldwide. The primary goal of the platform is to help travelers discover and access accommodations effortlessly, making trip planning more convenient, efficient, and user-friendly. Inspired by modern travel and accommodation platforms, Wanderlust offers an intuitive interface where users can browse, create, edit, and manage property listings with ease.

🛠️ Tech Stack & Features
🎨 Frontend
HTML5, CSS3, and JavaScript – Ensuring a responsive, interactive, and user-friendly interface.
Bootstrap – Enhancing design consistency, responsiveness, and overall user experience across multiple devices.
EJS & EJS-Mate – Enabling dynamic content rendering and reusable layouts.
AJAX – Providing asynchronous communication for smoother user interactions and dynamic content updates.
⚙️ Backend
Node.js – Providing a scalable and efficient server-side runtime environment.
Express.js – Managing routes, middleware, and server-side application logic.
RESTful APIs – Enabling efficient communication between the frontend and backend.
Express Sessions & Cookies – Managing user sessions and authentication securely.
Passport.js – Implementing user authentication and authorization mechanisms.
🗄️ Database
MongoDB – Serving as the primary NoSQL database for storing user and listing information.
Mongoose – Handling schema design, validation, middleware, and database interactions.
✨ Key Features
🌍 Browse hotel and accommodation listings from different cities worldwide.
👤 User Registration, Login, and Logout functionality.
🔐 Secure Authentication and Authorization.
🏠 Create, Edit, View, and Delete Listings (CRUD Operations).
⚡ Dynamic content loading using AJAX.
📝 Form and Schema Validation for data integrity.
⚠️ Custom Error Handling for improved reliability.
📱 Fully Responsive Design optimized for desktop, tablet, and mobile devices.
🏗️ Structured MVC Architecture for better code organization and maintainability.
