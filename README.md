# 🌍 Wanderlust – Full-Stack Hotel Listing Platform

## 📌 Project Overview

**Wanderlust** is a full-stack web application designed to provide a seamless hotel and accommodation listing experience across cities worldwide. The platform helps travelers discover, explore, and access accommodations effortlessly, making trip planning more convenient, efficient, and user-friendly.

Inspired by modern travel and booking platforms, Wanderlust offers an intuitive interface where users can browse, create, edit, and manage property listings with ease. The application follows the MVC architecture and incorporates secure authentication, responsive design, and robust backend functionality to deliver a smooth user experience.

---

## 🛠️ Tech Stack

### 🎨 Frontend

* **HTML5** – Structure and content of web pages.
* **CSS3** – Styling and responsive layouts.
* **JavaScript** – Interactive client-side functionality.
* **Bootstrap** – Responsive and mobile-first UI design.
* **EJS (Embedded JavaScript)** – Dynamic content rendering.
* **EJS-Mate** – Layout management and reusable templates.
* **AJAX** – Asynchronous requests for enhanced user experience.

### ⚙️ Backend

* **Node.js** – Server-side JavaScript runtime environment.
* **Express.js** – Web application framework for routing and middleware.
* **RESTful APIs** – Efficient communication between frontend and backend.
* **Express Sessions & Cookies** – Session and user state management.
* **Passport.js** – Secure authentication and authorization.

### 🗄️ Database

* **MongoDB** – NoSQL database for storing application data.
* **Mongoose** – ODM library for schema design, validation, and database operations.

## ✨ Key Features

### 🌍 Property Exploration

* Browse hotel and accommodation listings from different cities worldwide.
* View detailed property information, including images, descriptions, and pricing.

### 👤 User Authentication

* User Registration and Login System.
* Secure Logout Functionality.
* Authentication and Authorization using Passport.js.

### 🏠 Listing Management

* Create new accommodation listings.
* View listing details.

* Delete listings.

### ⚡ Dynamic User Experience

* AJAX-powered interactions for faster content updates.

### 📝 Data Validation & Error Handling

* Form validation for user inputs.
* Schema validation using Mongoose.
* Custom error handling and user-friendly error messages.

### 📱 Responsive Design

* Fully optimized for Desktop, Tablet, and Mobile devices.
* Consistent user experience across different screen sizes.

### 🏗️ Scalable Architecture

* Structured MVC (Model-View-Controller) Architecture.
* Modular and maintainable codebase.
* Separation of concerns for improved scalability and development efficiency.
